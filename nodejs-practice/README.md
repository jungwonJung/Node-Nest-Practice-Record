## 설치 방법

OS X & 리눅스:
1. Clone the repo
```sh
git clone https://github.com/JUNGganzi/nodejs_basic.git
```
2.  Install NPM packages 
```
npm init
```
3. install dependencies Modules
```
npm install
```
